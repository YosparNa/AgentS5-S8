# AI 口播脚本生成 — 技能与提示词整理

> 来源：网上搜集整理，涵盖主流写作模型、提示词模板、去AI味技巧。

---

## 一、经典写作模型（脚本结构框架）

### 1.1 SCQA 模型

适合：知识类、干货类、痛点类口播

| 要素 | 含义 | 示例 |
|------|------|------|
| **S** (Situation) | 情境 | "很多宝妈想给孩子做早餐..." |
| **C** (Complication) | 冲突/矛盾 | "但每天早起太累，根本没时间" |
| **Q** (Question) | 疑问 | "有没有5分钟就能搞定的办法？" |
| **A** (Answer) | 答案 | "教你3道懒人早餐，零失败！" |

### 1.2 AIDA 模型

适合：带货视频、种草视频、营销推广

| 要素 | 含义 | 作用 |
|------|------|------|
| **A** (Attention) | 吸引注意 | 开头抓眼球（前3秒） |
| **I** (Interest) | 激发兴趣 | 引发好奇/共鸣 |
| **D** (Desire) | 勾起欲望 | 展示好处/效果 |
| **A** (Action) | 促使行动 | 引导点赞/关注/购买 |

### 1.3 PAS 模型

适合：卖货、种草、解决问题类内容

| 要素 | 含义 | 示例 |
|------|------|------|
| **P** (Problem) | 痛点问题 | "皮肤暗沉发黄，怎么遮都遮不住" |
| **A** (Agitation) | 放大焦虑 | "相亲对象第一眼就看到你的暗沉..." |
| **S** (Solution) | 给出方案 | "这款精华，28天提亮一个色号" |

### 1.4 黄金3段式

最通用的口播结构：

```
开头（0-5秒）：钩子句
  → 反问句 / 数据冲击 / 痛点直击 / 颠覆认知

中间（5-45秒）：价值输出
  → 分点论述（3个点最佳）
  → 每个点 = 观点 + 案例/数据

结尾（45-60秒）：行动号召
  → 总结 + 引导互动（点赞/关注/评论）
```

### 1.5 时间占比参考

| 阶段 | 占比 | 内容 |
|------|------|------|
| 钩子（Hook） | 5-10% | 前3秒抓住注意力 |
| 痛点 | 20-30% | 共鸣感、场景化描述 |
| 方案/干货 | 40-50% | 核心内容，分点讲解 |
| 结果/案例 | 15-20% | 展示效果 |
| CTA | 5-10% | 引导互动 |

## 二、爆款开头公式

| 类型 | 公式 | 示例 |
|------|------|------|
| **冲突式** | 99%的人都不知道... | "99%的人都不知道这样喝水会伤肾" |
| **提问式** | 你是不是也...？ | "你是不是也每天洗头还是油？" |
| **反常识式** | 别再...了！ | "别再给孩子喝牛奶了！" |
| **数字式** | 只用了X招/只花了X天 | "只用了一招，涨粉10万" |
| **故事式** | 我曾经...直到... | "我曾经月薪3千，直到学会这个技能" |
| **权威式** | 作为一个XX，告诉你... | "做了10年皮肤科医生，告诉你真相" |
| **悬念式** | 千万别这样做，否则... | "千万别这样做，否则皮肤越来越差" |
| **利益式** | 教你免费... | "教你免费领取一份XX攻略" |

## 三、提示词模板

### 3.1 基础模板

```
你是一个专业的短视频口播脚本写手。请帮我写一个关于【主题】的口播脚本，要求：
1. 时长：60秒以内
2. 风格：轻松/专业/悬念式开头
3. 结构：痛点引入 → 核心内容 → 行动号召(CTA)
4. 目标受众：【描述受众】
5. 平台：抖音/小红书/视频号
```

### 3.2 结构化模板

```
【角色】你是一位资深短视频文案策划专家
【任务】请用【SCQA/AIDA/PAS】模型为以下主题写一条60秒口播脚本
【主题】：___________
【目标人群】：___________
【核心卖点/价值】：___________
【要求】：
  1. 开头3秒必须有强钩子（提问/反常识/数据冲击）
  2. 口语化表达，短句为主，不要书面语
  3. 字数控制在200-300字
  4. 结尾有明确的行动号召（CTA）
  5. 适当加入情绪词和口语化连接词
```

### 3.3 进阶模板（钩子-共鸣-干货-金句-CTA）

```
请用"钩子-共鸣-干货-金句-CTA"结构，为我写一个关于【主题】的短视频口播脚本。
要求：
- 开头用反常识句式制造冲突
- 中间用"第一/第二/第三"分点讲解
- 语言口语化，像朋友聊天
- 结尾用一句金句+引导互动
- 全文控制在300字以内
```

### 3.4 去AI味模板

```
你是一个短视频文案写手。请帮我写一条关于【主题】的短视频脚本。

要求：
1. 用大白话，口语化，像在跟朋友聊天
2. 禁止使用：赋能、底层逻辑、颠覆、值得注意的是
3. 第一句话就要抓住人，可以用反问/冲突/痛点切入
4. 加入具体数字、场景、细节（不是泛泛而谈）
5. 每句话控制在15字以内，方便口播
6. 结尾不要"升华"，用一个金句或反问收尾
7. 适当加入"你想想""说白了""这事儿就离谱"等口语词
```

### 3.5 分步骤生成模板

```
第一步：先列大纲
"请列出一个关于【主题】的短视频口播大纲，包含：钩子、3个核心要点、CTA"

第二步：逐段展开
"请把第1个要点展开成口语化的口播文字，15字以内短句，像朋友聊天"

第三步：润色
"请把以下文案改得更像一个真实的人在说话，加入口语化表达，删除书面语"
```

## 四、去AI味核心技巧

### 4.1 AI味特征 vs 人味替代

| AI味特征 | 人味替代 |
|---------|---------|
| "首先...其次...最后..." | 直接说重点，逻辑暗线 |
| "值得注意的是" | "说实话""你知道吗" |
| "在当今社会" | 直接切入具体场景 |
| 每段太工整 | 长短句混搭 |
| 高频使用"赋能""底层逻辑" | 用大白话解释 |
| 过度总结升华 | 戛然而止，留有余味 |
| 句句带"你" | 自然对话，不用每句点名 |

### 4.2 去AI味心法

1. **加入个人经历/情感** → "我上次试了一下，差点翻车"
2. **用具体代替抽象** → 不说"效率提升"，说"以前2小时，现在20分钟"
3. **制造不完美感** → 偶尔用"反正就是这么个意思"
4. **情绪先行** → 开头就带情绪（惊讶/愤怒/搞笑）
5. **加"废话"** → 适当加入看似多余但真实的语气词
6. **反套路** → AI爱写对称结构，人会突然转折

### 4.3 口语化词汇参考

**开头引导：** 你看啊——、说真的——、你知道吗——、听好了——、划重点、直接上干货

**过渡衔接：** 接下来、重点来了、还有一个关键的、对了、换个角度想

**互动感：** 你是不是也这样、你肯定遇到过、你觉得呢、评论区说说

**情绪表达：** 哎、天哪、绝了、说实话、就是那个……怎么说呢

**口语缓冲：** 怎么说呢、就是那个意思、你懂我意思吧、大概就是这么个情况

## 六、进阶技巧

### 6.1 多版本迭代法
1. 让AI生成3个不同风格的版本
2. 人工挑选最好的部分
3. 合并优化，再让AI润色
4. 重复2-3轮

### 6.2 喂范本法
给AI几个你觉得「有味道」的脚本作为参考：
```
以下是一个风格参考：【粘贴范本】
请模仿这个风格，写一个关于【主题】的口播脚本
```

### 6.3 角色扮演法
```
你现在是一个东北/四川/广东的博主，用方言感的普通话写一个关于【主题】的口播脚本
```

### 6.4 二次润色法
```
请把以下文案改得更像一个真实的人在说话，加入口语化表达，删除书面语
```

### 6.5 分步骤生成法
```
第一步：先列大纲（钩子、3个要点、CTA）
第二步：逐段展开成口语化文字
第三步：整体润色，去掉AI味
```

## 七、不同平台的口播风格差异

| 平台 | 风格特点 | 时长建议 |
|------|---------|---------|
| 抖音 | 快节奏、强钩子、情绪化 | 15-60秒 |
| B站 | 可以稍长、有深度、有观点 | 3-10分钟 |
| 小红书 | 种草感、生活化、有审美 | 30-90秒 |
| 视频号 | 偏中年受众、正能量、实用 | 1-3分钟 |
| YouTube | 可以很长、深度内容、结构清晰 | 5-20分钟 |
