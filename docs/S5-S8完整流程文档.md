## S5 选题

### 配置页
| 配置项 | 类型 | 说明 |
|--------|------|------|
| 频道定位描述 | textarea + 清除按钮 | 频道定位、受众、风格 |
| 输入数据 | textarea + 清除按钮 | 热点、竞品、评论、行业数据 |
| 反AI套路 | 3个开关 | 排除震惊体/假大空/AI套话 |
| 情绪波动评分 | 4个滑块 | 认知冲突/共鸣度/危机感/好奇驱动 |

### 产物页
5 个选题卡片，可点击选择，选中高亮

### 底部按钮
| 状态 | 按钮 |
|------|------|
| 未生成 | [AI 生成选题] |
| 已生成未锁定 | [AI 生成选题] + [锁定选题] |
| 已锁定 | [AI 生成选题(变淡)] + [解锁] |

### 锁定逻辑
- 点击选题卡片选择 → 点「锁定选题」
- 锁定后调用 `/api/s5/select` 设置后端 selected_topic
- 锁定后生成按钮变淡，需解锁才能重新生成
- S6 依赖检查：S5 必须锁定选题

### 输入框行为
- `oninput` 实时保存到全局变量 `savedUserData`/`savedChannelDesc`
- 切换 Tab 后自动恢复
- 每个输入框上方有「🗑 清除」按钮

## S6 大纲

### 配置页
| 配置项 | 类型 | 说明 |
|--------|------|------|
| 前3秒钩子 | 4选1 | 反常识/冲突悬念/数字冲击/画面承诺 |
| 节奏曲线 | 3个滑块 | 危机点密度/高潮位%/章节数 |

### 产物页
- **张力曲线**：渐变填充 + 白→红线性渐变色
- **章节卡片**：左侧彩色边框（按张力值变色）+ 圆形序号
  - 危机点：红色边框 + 红底白字 `⚠ 危机点` 标签
  - 张力颜色：1-4浅灰、5靛蓝、6天蓝、7翠绿、8金黄、9亮橙、10正红
- **详情按钮**：展开显示 pros/cons
- **改进/重做按钮**：
  - 有 cons → 显示「改进」（自动用 cons 作为反馈）
  - 无 cons → 显示「重做」（全新生成）
  - 生成中 → 锁定不可点击
- **张力曲线交互**：悬浮显示章节信息，点击跳转到对应卡片

### 单独章节重做
- 调用 `POST /api/s6/regen-chapter`
- `feedback` 为空 = 全新重做
- `feedback` 有 cons = 改进模式
- 使用 `regeneratingChapters` Set 防重复

### 底部按钮
| 状态 | 按钮 |
|------|------|
| 未生成 | [AI 生成大纲] |
| 已生成 | [AI 生成大纲] |

## S8 对抗审核

### 配置页
| 配置项 | 类型 | 说明 |
|--------|------|------|
| 红队角色 | 5个开关 | 杠精/同行/小白/老粉/合规 |

### 产物页
5 角色卡片 + 评分 + 问题列表 + 综合评分

### 底部按钮
| 状态 | 按钮 |
|------|------|
| 未生成 | [启动5角色对抗审核] |
| 已生成 | [重新审核] |
| 有回滚 | [执行回滚] + [重新审核] |

### 回滚逻辑
- 平均分 ≥ 7 → 通过
- 平均分 5-7 → 回滚到 S7
- 平均分 < 5 → 回滚到 S5

## 运行历史

| 功能 | 说明 |
|------|------|
| 自动保存 | 每次成功生成后自动保存，保存的是产物页格式的数据 |
| 查看记录 | 按时间倒序展示 |
| 当前标记 | 通过 `stageCurrentHist` 追踪，回滚切换标记不插入新记录 |
| 回滚 | 点击历史记录的「回滚」按钮，切换当前标记 + 恢复数据到产物页 |
| 数据一致性 | 保存格式与 STAGE.output 格式一致（S7 保存 `{excerpt}` 而非 `{script}`） |
| 最多 20 条 | 超出自动删除最旧记录 |

## 通用规则

### 生成按钮逻辑
- 未生成 → [AI 生成xxx]
- 已生成 → 可再次点击重新生成

### 失败保护
- 生成失败时保留旧数据不丢失（备份/恢复机制）
- 历史记录不受失败影响

### 依赖检查
- S6 检查 S5 锁定 → "请先在 S5 锁定选题"
- S7 检查 S6 数据 → "请先在 S6 生成大纲"
- S8 检查 S7 数据 → "请先在 S7 生成脚本"

### 防重复
- `agentLock` 全局锁，同一时间只有一个阶段运行
- `regeneratingChapters` Set 防止章节重做重复触发

### 提示系统
- Toast：居中、渐变、1.2秒自动消失、新弹覆盖旧弹
- 确认弹窗：居中、取消/确定按钮
